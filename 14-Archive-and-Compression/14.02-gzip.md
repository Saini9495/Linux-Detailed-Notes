# gzip Command

## Introduction

gzip is used to compress files in Linux.

It reduces file size and saves storage space.

## Why Use gzip?

- Save disk space
- Faster file transfer
- Compress log files

## Syntax

```bash
gzip filename
```

## Examples

### Compress File

```bash
gzip file.txt
```

Output:

```bash
file.txt.gz
```

### Decompress File

```bash
gunzip file.txt.gz
```

### Keep Original File

```bash
gzip -k file.txt
```

### View Compression Info

```bash
gzip -l file.txt.gz
```

## Common Options

| Option | Description |
|---------|-------------|
| -k | Keep original |
| -d | Decompress |
| -l | List info |
| -r | Recursive |

## Real World Use Cases

### Compress Logs

```bash
gzip access.log
```

### Compress Backup

```bash
gzip backup.tar
```

Result:

```bash
backup.tar.gz
```

## Interview Questions

### Q1 Difference between tar and gzip?

Tar archives files.

Gzip compresses files.

### Q2 What is .tar.gz?

A tar archive compressed using gzip.

## Summary

- Reduces file size
- Common with backups
- Creates .gz files
