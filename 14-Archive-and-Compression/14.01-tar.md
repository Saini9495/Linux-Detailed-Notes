# tar Command

## Introduction

The `tar` command stands for Tape Archive.

It is used to combine multiple files and directories into a single archive file.

## Why We Use tar?

- Create backups
- Transfer multiple files as one file
- Compress directories
- Store logs

## Syntax

```bash
tar [options] archive-name files
```

## Common Options

| Option | Description |
|----------|-------------|
| -c | Create archive |
| -x | Extract archive |
| -v | Verbose output |
| -f | Archive file name |
| -t | List archive contents |

## Examples

### Create Archive

```bash
tar -cvf backup.tar myfolder
```

### Extract Archive

```bash
tar -xvf backup.tar
```

### View Contents

```bash
tar -tvf backup.tar
```

### Archive Multiple Files

```bash
tar -cvf files.tar file1.txt file2.txt file3.txt
```

## Real World Use Cases

### Backup Home Directory

```bash
tar -cvf home_backup.tar /home/ubuntu
```

### Backup Log Files

```bash
tar -cvf logs.tar /var/log
```

## Interview Questions

### Q1 What is tar?

A utility used to archive multiple files into one file.

### Q2 Does tar compress files?

No. Tar only archives. Compression requires gzip or bzip2.

## Summary

- Archive files
- Combine directories
