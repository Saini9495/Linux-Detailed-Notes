# zip and unzip Commands

## Introduction

zip is used to compress files and directories.

unzip is used to extract zip files.

## Syntax

### Create Zip

```bash
zip archive.zip file.txt
```

### Extract Zip

```bash
unzip archive.zip
```

## Examples

### Compress Single File

```bash
zip data.zip file.txt
```

### Compress Multiple Files

```bash
zip files.zip file1.txt file2.txt file3.txt
```

### Compress Folder

```bash
zip -r project.zip project/
```

### Extract Zip

```bash
unzip project.zip
```

### Extract to Specific Location

```bash
unzip project.zip -d /home/ubuntu/extracted
```

## Common Options

| Option | Description |
|---------|-------------|
| -r | Recursive |
| -d | Extract path |
| -l | List contents |

## Real World Use Cases

### Share Project

```bash
zip -r website.zip website/
```

### Download and Extract

```bash
unzip project.zip
```

## Interview Questions

### Q1 Why zip instead of tar?

Zip works across Linux, Windows, and Mac.

### Q2 What does -r mean?

Recursive compression.

## Summary

- Easy compression
- Cross-platform support
