# Absolute Path vs Relative Path

## What is a Path?

A path is the location of a file or directory.

---

# Absolute Path

Starts from Root Directory (/).

Example:

```bash
/home/rishabh/Documents/file.txt
```

Characteristics:

- Complete path
- Starts with /
- Works from any location

---

# Relative Path

Path relative to current directory.

Example:

Current Directory:

```bash
/home/rishabh
```

Access file:

```bash
Documents/file.txt
```

No need to start from root.

---

# Examples

## Current Directory

```bash
pwd
```

Output:

```bash
/home/rishabh
```

### Absolute Path

```bash
cat /home/rishabh/Documents/file.txt
```

### Relative Path

```bash
cat Documents/file.txt
```

---

# Special Symbols

## Current Directory

```bash
.
```

## Parent Directory

```bash
..
```

Example:

```bash
cd ..
```

Move one directory up.

---

# Comparison Table

| Feature | Absolute Path | Relative Path |
|----------|---------------|---------------|
| Starts with / | Yes | No |
| Full Location | Yes | No |
| Depends on Current Directory | No | Yes |
| Easier in Scripts | Yes | Sometimes |

---
