# Linux File System

## What is Linux File System?

Linux File System is the method used by Linux operating systems to organize, store, and retrieve files and directories.

Unlike Windows, Linux does not use drive letters such as C:, D:, etc.

Everything in Linux starts from a single root directory:

```bash
/
```

This is called the Root Directory.

## Key Features

- Hierarchical Structure
- Everything is a File
- Case Sensitive
- Secure Permission Model
- Supports Multiple File Systems

## File System Hierarchy

```text
/
├── bin
├── boot
├── dev
├── etc
├── home
├── lib
├── media
├── mnt
├── opt
├── proc
├── root
├── run
├── sbin
├── srv
├── sys
├── tmp
├── usr
└── var
```

## Check File System

```

## Check Mounted File Systems

```

## Advantages

- Better Security

- Efficient Resource Management
- Supports Multiple Users
