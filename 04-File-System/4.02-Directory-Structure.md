# Linux Directory Structure

## Root Directory (/)

All directories originate from root.

```bash
/
```

---

## /bin

Essential user commands.

Examples:

```bash
ls
cp
mv
cat
```

---

## /boot

Boot loader files and Linux kernel.

Example:

```bash
vmlinuz
grub
```

---

## /dev

Device files.

Examples:

```bash
/dev/sda
/dev/null
```

---

## /etc

System configuration files.

Examples:

```bash
passwd
hosts
ssh
```

---

## /home

Home directories for normal users.

Example:

```bash
/home/rishabh
```

---

## /root

Home directory of root user.

```bash
/root
```

---

## /tmp

Temporary files.

```bash
/tmp
```

Files may be deleted automatically after reboot.

---

## /var

Variable data.

Contains:

- Logs
- Cache
- Mail

Example:

```bash
/var/log
```

---

## /usr

User applications and utilities.

Contains:

```bash
/usr/bin
/usr/sbin
```

---

## /opt

Optional third-party software.

Example:

```bash
Google Chrome
Oracle Software
```

---

## Directory Structure Diagram

```text
/
├── bin
├── boot
├── dev
├── etc
├── home
│   └── user
├── root
├── tmp
├── usr
└── var
```

## Interview Questions

### Q1: Where are user home directories stored?

```bash
/home
```

### Q2: Which directory stores logs?

```bash
/var/log
```

### Q3: Which directory stores configuration files?

```bash
/etc
```
