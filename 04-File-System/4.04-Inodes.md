# Inodes in Linux

## What is an Inode?

An inode is a data structure that stores metadata about a file.

Metadata includes:

- File Size
- Owner
- Permissions
- Creation Time
- Modification Time
- File Type

An inode does NOT store:

- File Name
- Actual Data

---

## How Inode Works

```text
File Name
    |
    v
Inode Number
    |
    v
Actual Data Blocks
```

---

## View Inode Number

```bash
ls -i
```

Example Output:

```bash
12345 file.txt
```

Here:

```bash
12345
```

is the inode number.

---

## View File Details

```bash
stat file.txt
```

Output includes:

- Inode Number
- Permissions
- Size
- Timestamps

---

## Why Inodes are Important?

- File tracking
- Hard links
- Metadata storage
- File management

---

## Hard Link Example

Create Hard Link:

```bash
ln file.txt file2.txt
```

Check inode:

```bash
ls -i
```

Both files will have same inode number.

---

## Find Inode Usage

```bash
df -i
```

Output:

```bash
Filesystem Inodes IUsed IFree
```
