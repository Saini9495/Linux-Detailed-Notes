# SSH Basics

## What is SSH?

SSH (Secure Shell) is a secure network protocol used to connect to remote servers over an encrypted connection.

### Why SSH?

- Secure communication
- Remote server management
- File transfers
- Command execution on remote machines

---

## SSH Architecture

Client → SSH → Server

Example:

Your Laptop
     ↓
SSH Connection
     ↓
AWS EC2 Instance

---

## Default SSH Port

Port: 22

Check SSH Port:

```bash
sudo netstat -tulpn | grep ssh
```

---

## Install SSH

Ubuntu:

```bash
sudo apt update
sudo apt install openssh-server
```

Check Status:

```bash
sudo systemctl status ssh
```

Start Service:

```bash
sudo systemctl start ssh
```

Enable Service:

```bash
sudo systemctl enable ssh
```

---

## Connect to Remote Server

Syntax:

```bash
ssh username@server-ip
```

Example:

```bash
ssh ubuntu@13.233.xx.xx
```

---

## Check SSH Version

```bash
ssh -V
```

---

## Important SSH Commands

```bash
ssh user@host
exit
whoami
hostname
pwd
```
