# SCP (Secure Copy)

## What is SCP?

SCP is used to securely copy files between systems using SSH.

---

## Syntax

```bash
scp source destination
```

---

## Copy Local File to Remote Server

```bash
scp file.txt ubuntu@server-ip:/home/ubuntu
```

Example:

```bash
scp notes.txt ubuntu@13.233.xx.xx:/home/ubuntu
```

---

## Copy Remote File to Local Machine

```bash
scp ubuntu@13.233.xx.xx:/home/ubuntu/file.txt .
```

---

## Copy Directory

```bash
scp -r project ubuntu@server-ip:/home/ubuntu
```

---

## Use Private Key

```bash
scp -i mykey.pem file.txt ubuntu@server-ip:/home/ubuntu
```

---

## Change Port

```bash
scp -P 2222 file.txt ubuntu@server-ip:/home/ubuntu
```

---

## Common Options

```bash
-r
-P
-i
-v
```

---

## Real World Example

Deploy Website Files:

```bash
scp -r dist/* ubuntu@server-ip:/var/www/html
```
