# Rsync

## What is Rsync?

Rsync is a powerful file synchronization and transfer utility.

---

## Why Rsync?

- Faster than SCP
- Transfers only changed files
- Supports compression
- Ideal for backups

---

## Basic Syntax

```bash
rsync [options] source destination
```

---

## Copy Files

```bash
rsync file.txt backup/
```

---

## Copy Directory

```bash
rsync -av project/ backup/
```

---

## Sync Local to Remote

```bash
rsync -av project/ ubuntu@server-ip:/home/ubuntu/project
```

---

## Sync Remote to Local

```bash
rsync -av ubuntu@server-ip:/home/ubuntu/project .
```

---

## Compression

```bash
rsync -avz project/ ubuntu@server-ip:/home/ubuntu
```

---

## Delete Extra Files

```bash
rsync -av --delete source/ destination/
```

---

## Progress Bar

```bash
rsync -av --progress source/ destination/
```

---

## Useful Options

```bash
-a
-v
-z
--delete
--progress
```

---

## Real World Backup

```bash
rsync -avz /var/www/ backup-server:/backup/
```
