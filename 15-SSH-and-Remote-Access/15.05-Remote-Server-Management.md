# Remote Server Management

## Introduction

Remote Server Management means controlling and managing servers from another machine using SSH.

---

## Connect to Server

```bash
ssh ubuntu@server-ip
```

## Security Checklist

### Disable Root Login

Edit:

```bash
sudo nano /etc/ssh/sshd_config
```

Set:

```text
PermitRootLogin no
```

Restart SSH:

```bash
sudo systemctl restart ssh
```

---

### Disable Password Authentication

```text
PasswordAuthentication no
```

Restart:

```bash
sudo systemctl restart ssh
```
