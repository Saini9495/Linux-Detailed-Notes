# mv Command

## What is mv?

Move or rename files and directories.

---

## Syntax

```bash
mv source destination
```

---

## Examples

### Rename File

```bash
mv old.txt new.txt
```

### Move File

```bash
mv file.txt Documents/
```

### Move Directory

```bash
mv project backup/
```

---

## Use Cases

- Rename files
- Move files/folders
