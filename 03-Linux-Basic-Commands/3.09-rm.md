# rm Command

## What is rm?

Removes files and directories.

---

## Syntax

```bash
rm filename
```

---

## Examples

### Delete File

```bash
rm file.txt
```

### Delete Directory

```bash
rm -r project
```

### Force Delete

```bash
rm -rf project
```

---

## Warning

```bash
rm -rf /
```

Never execute this command.

It can delete the entire Linux system.
