# cd Command

## What is cd?

`cd` stands for Change Directory.

Used to move between directories.

---

## Syntax

```bash
cd directory_name
```

---

## Examples

### Move to Home

```bash
cd
```

### Move to Specific Directory

```bash
cd Documents
```

### Move Back One Directory

```bash
cd ..
```

### Move Two Directories Back

```bash
cd ../..
```

### Previous Directory

```bash
cd -
```

---

## Use Cases

- Navigate file system
- Access project folders
