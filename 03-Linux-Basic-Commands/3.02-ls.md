# ls Command

## What is ls?

`ls` is used to list files and directories.

---

## Syntax

```bash
ls
```

---

## Common Options

### List files

```bash
ls
```

### Detailed View

```bash
ls -l
```

### Hidden Files

```bash
ls -a
```

### Human Readable Size

```bash
ls -lh
```

### Sort by Time

```bash
ls -lt
```

---

## Example

```bash
ls -la
```

Output:

```bash
drwxr-xr-x user user
-rw-r--r-- file.txt
```
