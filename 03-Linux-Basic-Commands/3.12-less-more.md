# less and more Commands

## more

Displays file content page by page.

```bash
more file.txt
```

---

## less

Advanced version of more.

```bash
less file.txt
```

---

## Advantages of less

- Scroll up and down
- Search content
- Faster navigation

---

## Useful Keys

### Search

```bash
/search_term
```

### Quit

```bash
q
```
