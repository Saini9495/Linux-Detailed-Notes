# pwd Command

## What is pwd?

`pwd` stands for **Print Working Directory**.

It displays the absolute path of the current directory.

---

## Syntax

```bash
pwd
```

---

## Example

```bash
pwd
```

Output:

```bash
/home/ubuntu
```

---

## Use Case

- Find current directory location
- Useful while navigating directories

---

## Interview Question

### Q. What does pwd command do?

It displays the absolute path of the current working directory.
