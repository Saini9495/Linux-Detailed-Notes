# cat Command

## What is cat?

Used to display file contents.

---

## Syntax

```bash
cat filename
```

---

## Examples

### Read File

```bash
cat notes.txt
```

### Create File

```bash
cat > file.txt
```

### Merge Files

```bash
cat file1 file2 > combined.txt
```

---

## Use Cases

- Read logs
- View configuration files
