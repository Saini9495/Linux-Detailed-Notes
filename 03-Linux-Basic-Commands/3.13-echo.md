# echo Command

## What is echo?

Displays text on terminal.

---

## Syntax

```bash
echo "Hello Linux"
```

---

## Examples

### Print Text

```bash
echo "Welcome"
```

### Store Content

```bash
echo "DevOps" > file.txt
```

### Append Content

```bash
echo "Docker" >> file.txt
```

### Display Variables

```bash
echo $HOME
```

---

## Use Cases

- Shell scripting
- Debugging
- Writing content to files
