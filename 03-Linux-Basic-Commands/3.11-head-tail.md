# head and tail Commands

## head

Displays first lines of a file.

### Syntax

```bash
head file.txt
```

### First 5 Lines

```bash
head -5 file.txt
```

---

## tail

Displays last lines of a file.

### Syntax

```bash
tail file.txt
```

### Last 20 Lines

```bash
tail -20 file.txt
```

### Live Logs

```bash
tail -f /var/log/syslog
```

---

## DevOps Usage

Monitoring logs in real-time.

```bash
tail -f application.log
```
