# cp Command

## What is cp?

Copy files and directories.

---

## Syntax

```bash
cp source destination
```

---

## Examples

### Copy File

```bash
cp file.txt backup.txt
```

### Copy Directory

```bash
cp -r project backup
```

---

## Important Options

### Recursive

```bash
cp -r
```

### Interactive

```bash
cp -i
```

### Preserve Metadata

```bash
cp -p
```
