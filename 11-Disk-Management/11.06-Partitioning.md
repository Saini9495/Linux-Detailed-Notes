# Disk Partitioning

## Introduction

Partitioning divides a physical disk into logical sections.

## Why Partitioning?

- Better storage management
- Separate OS and data
- Enhanced security
- Easy backup and recovery

## Partition Types

### Primary Partition

Main partition used for OS installation.

### Extended Partition

Container for logical partitions.

### Logical Partition

Created inside extended partition.

## MBR vs GPT

| MBR | GPT |
|-------|-------|
| 2 TB limit | Supports larger disks |
| 4 primary partitions | Up to 128 partitions |
| Older system | Modern system |

## Partitioning Workflow

### Step 1

Check disks

```bash
lsblk
```

### Step 2

Create partition

```bash
sudo fdisk /dev/sdb
```

### Step 3

Format partition

```bash
sudo mkfs.ext4 /dev/sdb1
```

### Step 4

Mount partition

```bash
sudo mount /dev/sdb1 /mnt/data
```

## Practical Use Cases

- AWS EBS setup
- Database storage separation
- Application data storage

## Interview Questions

### Q1. Difference between MBR and GPT?
