# fdisk Command

## Introduction

fdisk is a disk partition management utility.

## Why We Use It

- Create partitions
- Delete partitions
- Modify partition tables

## Syntax

```bash
sudo fdisk /dev/sda
```

## Common Commands Inside fdisk

| Command | Description |
|----------|-------------|
| p | Print partitions |
| n | New partition |
| d | Delete partition |
| w | Write changes |
| q | Quit |

## Examples

### List partitions

```bash
sudo fdisk -l
```

### Create partition

```bash
sudo fdisk /dev/sda
```

## Practical Use Cases

- New disk setup
- Storage expansion
- Linux server administration

## Interview Questions

### Q1. What does fdisk do?

It manages disk partitions.

## Summary

fdisk is used for partition creation and management.
