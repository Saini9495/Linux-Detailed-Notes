# df Command

## Introduction
The `df` (Disk Filesystem) command is used to display disk space usage of mounted file systems.

## Why We Use It
- Check available disk space
- Monitor storage usage
- Troubleshoot low disk issues

## Syntax

```bash
df [OPTION] [FILE]
```

## Common Options

| Option | Description |
|----------|-------------|
| -h | Human readable format |
| -T | Show file system type |
| -a | Show all file systems |
| -i | Show inode information |

## Examples

### Check disk usage

```bash
df
```

### Human readable output

```bash
df -h
```

### Show filesystem type

```bash
df -Th
```

## Practical Use Cases

- Checking EC2 disk utilization
- Monitoring server storage
- Troubleshooting "No space left on device"

## Interview Questions

### Q1. Difference between df and du?

df shows filesystem usage.
du shows directory/file usage.

## Summary

df is used to check available and used disk space.
