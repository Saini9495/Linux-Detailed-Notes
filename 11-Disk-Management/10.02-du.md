# du Command

## Introduction

The `du` (Disk Usage) command displays file and directory space usage.

## Why We Use It

- Find large directories
- Analyze storage consumption
- Clean unwanted files

## Syntax

```bash
du [OPTION] [FILE]
```

## Common Options

| Option | Description |
|----------|-------------|
| -h | Human readable |
| -s | Summary only |
| -a | Show files and directories |
| -c | Total size |

## Examples

### Directory size

```bash
du -h
```

### Specific folder

```bash
du -sh /var/log
```

### Total size

```bash
du -sch *
```

## Practical Use Cases

- Identify large log files
- Check Docker storage
- Server cleanup

## Interview Questions

### Q1. How to find largest directory?

```bash
du -sh * | sort -hr
```

## Summary

du helps analyze storage usage of files and directories.
