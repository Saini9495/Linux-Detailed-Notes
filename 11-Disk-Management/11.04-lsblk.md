# lsblk Command

## Introduction

lsblk lists all available block storage devices.

## Why We Use It

- View disks
- View partitions
- Identify mounted devices

## Syntax

```bash
lsblk
```

## Common Options

| Option | Description |
|----------|-------------|
| -f | Show filesystem info |
| -o | Custom output |
| -a | Show all devices |

## Examples

### List block devices

```bash
lsblk
```

### Filesystem details

```bash
lsblk -f
```

## Sample Output

```bash
NAME   SIZE TYPE MOUNTPOINT
xvda    20G disk
├─xvda1 19G part /
└─xvda2  1G part /boot
```

## Practical Use Cases

- EC2 disk identification
- Mount troubleshooting
- Partition verification

## Interview Questions

### Q1. Difference between lsblk and fdisk?

lsblk displays device hierarchy.
fdisk manages partitions.

## Summary

lsblk provides a quick overview of storage devices.
