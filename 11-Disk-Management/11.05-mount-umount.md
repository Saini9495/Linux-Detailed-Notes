# mount and umount Commands

## Introduction

mount attaches a filesystem to Linux directory tree.

umount detaches it.

## Why We Use It

- Access external drives
- Attach new disks
- Manage storage devices

## Syntax

### Mount

```bash
mount DEVICE DIRECTORY
```

### Unmount

```bash
umount DEVICE
```

## Examples

### Mount disk

```bash
sudo mount /dev/xvdf /mnt/data
```

### Verify mount

```bash
df -h
```

### Unmount disk

```bash
sudo umount /mnt/data
```

## Practical Use Cases

- AWS EBS attachment
- USB mounting
- Backup storage access

## Interview Questions

### Q1. Why can't a mounted disk be removed?

Because filesystem is in use.

## Summary

mount connects storage devices, umount disconnects them.
