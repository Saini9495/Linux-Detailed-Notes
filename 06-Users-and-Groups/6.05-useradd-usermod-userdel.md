# useradd, usermod and userdel Commands

## useradd

Used to create new users.

### Create User

```bash
sudo useradd username
```

### Create User with Home Directory

```bash
sudo useradd -m username
```

### Set Password

```bash
sudo passwd username
```

---

## usermod

Used to modify existing users.

### Change Username

```bash
sudo usermod -l newname oldname
```

### Add User to Group

```bash
sudo usermod -aG developers username
```

### Change Home Directory

```bash
sudo usermod -d /new/home username
```

### Lock User

```bash
sudo usermod -L username
```

### Unlock User

```bash
sudo usermod -U username
```

---

## userdel

Used to delete users.

### Delete User

```bash
sudo userdel username
```

### Delete User with Home Directory

```bash
sudo userdel -r username
```

---

## Complete Example

### Create User

```bash
sudo useradd -m devuser
```

### Set Password

```bash
sudo passwd devuser
```

### Add User to Group

```bash
sudo usermod -aG developers devuser
```

### Delete User

```bash
sudo userdel -r devuser
```

## Interview Questions

### Difference between useradd and adduser?

- useradd → low-level command
- adduser → user-friendly interactive command

### Difference between userdel and userdel -r?

- userdel → removes user only
- userdel -r → removes user and home directory
