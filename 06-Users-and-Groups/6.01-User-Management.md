# User Management in Linux

## What is a User?

A user is an account that can log in and access system resources.

## Types of Users

### 1. Root User
- Superuser of Linux
- Has full access to the system
- User ID (UID) = 0

### 2. System Users
- Used by services and applications
- Cannot normally log in

### 3. Regular Users
- Created for normal users
- Limited permissions

## Important Files

### /etc/passwd
Stores user account information.

```bash
cat /etc/passwd
```

### /etc/shadow
Stores encrypted passwords.

```bash
sudo cat /etc/shadow
```

## Display Current User

```bash
whoami
```

## Check Logged-In Users

```bash
who
```

```bash
w
```

## Check User ID

```bash
id
```

## Home Directory

```bash
echo $HOME
```

## Best Practices

- Use regular users for daily tasks.
- Avoid working directly as root.
- Give sudo access only when required.
