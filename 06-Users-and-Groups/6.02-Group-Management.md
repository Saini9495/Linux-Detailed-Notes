# Group Management in Linux

## What is a Group?

A group is a collection of users.

Groups help manage permissions efficiently.

## Check Current User Groups

```bash
groups
```

## Check Group Information

```bash
cat /etc/group
```

## Create a Group

```bash
sudo groupadd developers
```

## Delete a Group

```bash
sudo groupdel developers
```

## Add User to Group

```bash
sudo usermod -aG developers username
```

## Remove User from Group

```bash
sudo gpasswd -d username developers
```

## Check User Groups

```bash
id username
```

## Why Groups?

Example:
- Developers Group
- Testers Group
- Admins Group

Each group gets different permissions.
