# Sudo Command in Linux

## What is sudo?

sudo = Super User DO

Allows a normal user to execute commands with root privileges.

## Syntax

```bash
sudo command
```

## Example

```bash
sudo apt update
```

## Switch to Root User

```bash
sudo su
```

or

```bash
sudo -i
```

## Check sudo Permissions

```bash
sudo -l
```

## Edit sudoers File

```bash
sudo visudo
```

## Example Entry

```text
username ALL=(ALL:ALL) ALL
```

## Why Use sudo?

- Better security
- Tracks command usage
- Avoids direct root login
