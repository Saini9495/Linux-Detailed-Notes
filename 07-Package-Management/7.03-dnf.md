# DNF (Dandified YUM)

## What is DNF?

DNF is the modern replacement for YUM.

Used in:

- Fedora
- CentOS Stream
- RHEL 8+
- Rocky Linux
- AlmaLinux

---

## Install Package

```bash
sudo dnf install nginx
```

---

## Update Packages

```bash
sudo dnf update
```

---

## Search Package

```bash
dnf search nginx
```

---

## Package Information

```bash
dnf info nginx
```

---

## Remove Package

```bash
sudo dnf remove nginx
```

---

## List Installed Packages

```bash
dnf list installed
```

---

## Clean Cache

```bash
sudo dnf clean all
```

---

## Advantages of DNF

- Faster dependency handling
- Better performance
- Lower memory usage
- Improved package management
