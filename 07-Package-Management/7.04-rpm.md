# RPM (Red Hat Package Manager)

## What is RPM?

RPM is a package format used in Red Hat based distributions.

Examples:

- RHEL
- CentOS
- Rocky Linux
- AlmaLinux
- Fedora

Package Extension:

```text
.rpm
```

---

## Install RPM Package

```bash
sudo rpm -ivh package.rpm
```

Options:

- i = install
- v = verbose
- h = progress bar

---

## Upgrade Package

```bash
sudo rpm -Uvh package.rpm
```

---

## Remove Package

```bash
sudo rpm -e package-name
```

---

## Verify Installed Package

```bash
rpm -V package-name
```

---

## List Installed Packages

```bash
rpm -qa
```

---

## Package Information

```bash
rpm -qi package-name
```

---

## Interview Question

### Difference between RPM and YUM/DNF?

RPM installs packages directly.

YUM/DNF automatically resolve dependencies.
