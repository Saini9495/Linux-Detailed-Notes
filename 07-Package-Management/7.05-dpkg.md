# DPKG (Debian Package)

## What is DPKG?

DPKG is a low-level package manager used in Debian-based systems.

Package Extension:

```text
.deb
```

Examples:

- Ubuntu
- Debian
- Linux Mint
- Kali Linux

---

## Install Package

```bash
sudo dpkg -i package.deb
```

---

## Remove Package

```bash
sudo dpkg -r package-name
```

---

## List Installed Packages

```bash
dpkg -l
```

---

## Package Information

```bash
dpkg -s package-name
```

---

## Find Package Owner of File

```bash
dpkg -S /path/to/file
```

---

## Common Issue

Dependency errors can occur while installing packages.

Fix using:

```bash
sudo apt --fix-broken install
```

---

## Interview Question

### Difference between APT and DPKG?

DPKG is low-level and works directly with .deb files.

APT is high-level and automatically manages dependencies.
