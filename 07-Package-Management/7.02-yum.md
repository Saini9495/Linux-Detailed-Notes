# YUM (Yellowdog Updater Modified)

## What is YUM?

YUM is a package manager used in:

- CentOS 7
- RHEL 7
- Amazon Linux 1

---

## Install Package

```bash
sudo yum install nginx
```

---

## Update Packages

```bash
sudo yum update
```

---

## Search Package

```bash
yum search nginx
```

---

## Package Information

```bash
yum info nginx
```

---

## List Installed Packages

```bash
yum list installed
```

---

## Remove Package

```bash
sudo yum remove nginx
```

---

## Clean Cache

```bash
sudo yum clean all
```

---

## Interview Question

### Why was YUM replaced by DNF?

DNF provides better dependency resolution and performance.
