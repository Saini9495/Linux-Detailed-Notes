# Ubuntu Installation Guide

## What is Ubuntu?

Ubuntu is a Debian-based Linux distribution widely used for servers, cloud computing, and DevOps environments.

## System Requirements

- Processor: 2 GHz Dual Core
- RAM: 4 GB (Minimum)
- Storage: 25 GB
- USB Drive: 8 GB+

## Download Ubuntu ISO

Official Website:
https://ubuntu.com/download

## Create Bootable USB

### Using Rufus (Windows)

1. Download Rufus
2. Insert USB Drive
3. Select Ubuntu ISO
4. Click Start

## Installation Steps

1. Boot from USB
2. Select Install Ubuntu
3. Choose Keyboard Layout
4. Select Normal Installation
5. Choose Installation Type
6. Create User Account
7. Wait for Installation
8. Restart System

## Verify Installation

```bash
uname -a
```

```bash
lsb_release -a
```

## Update Ubuntu

```bash
sudo apt update
sudo apt upgrade -y
```

## Conclusion

Ubuntu is now successfully installed and ready for Linux learning.
