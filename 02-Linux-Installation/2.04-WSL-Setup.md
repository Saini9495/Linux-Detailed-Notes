# Windows Subsystem for Linux (WSL)

## What is WSL?

WSL allows Linux to run directly on Windows without using a virtual machine.

## Benefits

- Lightweight
- Fast Startup
- Direct Windows Integration
- Ideal for DevOps Learning

## Check WSL Status

Open PowerShell as Administrator

```powershell
wsl --status
```

## Install WSL

```powershell
wsl --install
```

Restart your computer.

## Install Ubuntu

```powershell
wsl --install -d Ubuntu
```

## Verify Installation

```powershell
wsl -l -v
```

## Open Ubuntu

```powershell
wsl
```

or

```powershell
ubuntu
```

## Update Packages

```bash
sudo apt update

sudo apt upgrade -y
```

## Useful WSL Commands

### List Distributions

```powershell
wsl -l
```

### Shutdown WSL

```powershell
wsl --shutdown
```

### Check Version

```powershell
wsl --version
```
