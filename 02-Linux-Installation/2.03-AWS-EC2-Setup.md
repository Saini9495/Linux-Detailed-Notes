# AWS EC2 Setup

## What is EC2?

Amazon EC2 (Elastic Compute Cloud) provides virtual servers in the cloud.

## Prerequisites

- AWS Account
- Free Tier Eligible Account

## Launch EC2 Instance

### Step 1: Login AWS Console

AWS Console → EC2

### Step 2: Launch Instance

Click Launch Instance

### Step 3: Configure

Name: Linux-Server

AMI: Ubuntu Server

Instance Type: t2.micro (Free Tier)

### Step 4: Create Key Pair

Create New Key Pair

Download .pem file

### Step 5: Configure Security Group

Allow:

- SSH (22)
- HTTP (80)
- HTTPS (443)

### Step 6: Launch Instance

Click Launch

## Connect to EC2

### Windows PowerShell

```bash
ssh -i key.pem ubuntu@public-ip
```

### Linux Terminal

```bash
chmod 400 key.pem

ssh -i key.pem ubuntu@public-ip
```

## Update Server

```bash
sudo apt update

sudo apt upgrade -y
```

## Verify Instance

```bash
hostname

uname -a
```

## Common EC2 Commands

### Check Memory

```bash
free -h
```

### Check Disk

```bash
df -h
```

### Check CPU

```bash
lscpu
```

EC2 Ubuntu server is ready for Linux and DevOps practice.
