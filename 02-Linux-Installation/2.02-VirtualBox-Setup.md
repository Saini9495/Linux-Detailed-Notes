# VirtualBox Setup for Linux

## What is VirtualBox?

VirtualBox is a free virtualization software used to run multiple operating systems on a single machine.

## Download VirtualBox

Official Website:
https://www.virtualbox.org

## Installation Steps

### Windows

1. Download VirtualBox Installer
2. Run Installer
3. Click Next
4. Complete Installation

## Create Ubuntu Virtual Machine

### Step 1: Open VirtualBox

Click New

### Step 2: Configure VM

Name: Ubuntu

Type: Linux

Version: Ubuntu (64-bit)

### Step 3: Allocate Resources

RAM: 4096 MB

CPU: 2 Cores

Disk: 25 GB (Dynamic)

### Step 4: Attach ISO

Settings → Storage

Select Ubuntu ISO

### Step 5: Start VM

Click Start

Ubuntu Installer will launch.

## Useful VirtualBox Features

### Snapshots

Save VM state

### Clone VM

Duplicate existing VM

### Shared Clipboard

Enable copy-paste between Host and VM

### Shared Folders

Share files between systems

## Verify Setup

```bash
hostnamectl
```

Ubuntu Virtual Machine is successfully configured using VirtualBox.
