# chmod Command

## What is chmod?

`chmod` stands for Change Mode.

It is used to change file and directory permissions in Linux.

## Syntax

```bash
chmod [permissions] filename
```

## Permission Values

| Permission | Value |
|------------|--------|
| Read (r)   | 4 |
| Write (w)  | 2 |
| Execute (x)| 1 |

## Examples

### Give Read, Write and Execute Permission

```bash
chmod 777 file.txt
```

### Give Read and Write Permission

```bash
chmod 666 file.txt
```

### Owner Full Permission Only

```bash
chmod 700 file.txt
```

### Add Execute Permission

```bash
chmod +x script.sh
```

### Remove Write Permission

```bash
chmod -w file.txt
```

## Common chmod Values

| Value | Meaning |
|---------|---------|
| 777 | Everyone has full access |
| 755 | Owner full, others read & execute |
| 700 | Owner only |
| 644 | Owner read/write, others read |
| 600 | Owner read/write only |

## Best Practices

- Avoid using 777.
- Use 755 for scripts.
- Use 644 for configuration files.
