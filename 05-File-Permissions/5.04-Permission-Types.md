# Linux Permission Types

Linux permissions control access to files and directories.

## Permission Categories

### Owner (u)

The creator of the file.

### Group (g)

Users belonging to the same group.

### Others (o)

Everyone else.

## Permission Types

| Symbol | Meaning |
|----------|---------|
| r | Read |
| w | Write |
| x | Execute |

## Numeric Representation

| Permission | Value |
|------------|--------|
| Read | 4 |
| Write | 2 |
| Execute | 1 |

## Examples

### 777

```text
rwx rwx rwx
```

### 755

```text
rwx r-x r-x
```

### 644

```text
rw- r-- r--
```

## Check Permissions

```bash
ls -l
```

Example:

```bash
-rwxr-xr-x
```

Breakdown:

```text
Owner  = rwx
Group  = r-x
Others = r-x
```
