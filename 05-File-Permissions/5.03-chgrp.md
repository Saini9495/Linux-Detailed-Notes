# chgrp Command

## What is chgrp?

`chgrp` stands for Change Group.

It changes the group ownership of files and directories.

## Syntax

```bash
chgrp group_name filename
```

## Examples

### Change Group

```bash
sudo chgrp developers file.txt
```

### Recursive Group Change

```bash
sudo chgrp -R developers project/
```

## Verify Group

```bash
ls -l
```

Output:

```bash
-rw-r--r-- 1 ubuntu developers 100 Jun 10 file.txt
```

## Why Use chgrp?

- Team collaboration
- Shared directories
- Permission management
