# Firewall in Linux

## What is a Firewall?

A Firewall is a security system that monitors and controls incoming and outgoing network traffic based on predefined security rules.

It acts as a barrier between a trusted internal network and untrusted external networks.

---

## Why Firewall is Important?

- Prevents unauthorized access
- Blocks malicious traffic
- Protects servers from attacks
- Controls network communication

---

## Types of Firewalls

### 1. Network Firewall
Protects an entire network.

### 2. Host-Based Firewall
Protects a single machine.

Examples:
- UFW
- firewalld
- iptables

---

## Linux Firewall Tools

### iptables
Traditional Linux firewall.

### nftables
Modern replacement for iptables.

### UFW
User Friendly Firewall.

### firewalld
Dynamic firewall manager.

---

## Firewall Workflow

Internet
↓
Firewall Rules
↓
Allow / Deny Traffic
↓
Linux Server

---

## Common Firewall Tasks

### Allow SSH

```bash
sudo ufw allow 22
```

### Allow HTTP

```bash
sudo ufw allow 80
```

### Allow HTTPS

```bash
sudo ufw allow 443
```

### Block a Port

```bash
sudo ufw deny 8080
```

---

## Best Practices

- Allow only required ports
- Disable unused services
- Monitor firewall logs
- Regularly audit firewall rules
- Restrict SSH access
