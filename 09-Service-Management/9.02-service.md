# Service Command

## Introduction

The `service` command is an older way to manage services in Linux.

## Syntax

```bash
service <service-name> <action>
```

## Common Examples

### Start Service

```bash
service nginx start
```

### Stop Service

```bash
service nginx stop
```

### Restart Service

```bash
service nginx restart
```

### Check Status

```bash
service nginx status
```

## Difference Between service and systemctl

| service | systemctl |
|----------|-----------|
| Old SysVinit | Modern systemd |
| Limited features | Advanced features |
| Simple syntax | More control |

## Example

```bash
sudo service apache2 restart
```

## Summary

- Legacy service management command
- Mostly replaced by systemctl
- Still available in many Linux distributions
