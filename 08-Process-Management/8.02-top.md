# TOP Command

## What is top?

The `top` command displays real-time system information including CPU, Memory, and Running Processes.

## Syntax

```bash
top
```

## Features

- Real-time monitoring
- CPU Usage
- Memory Usage
- Process Information

## Useful Shortcuts

| Key | Action |
|-------|---------|
| q | Quit |
| k | Kill Process |
| P | Sort by CPU |
| M | Sort by Memory |

## Example

```bash
top
```

## Real-Life Use Case

Monitor server performance during application deployment.
