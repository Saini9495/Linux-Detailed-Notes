# Nice and Renice Commands

## What is Nice?

The `nice` command starts a process with a specified priority.

## Priority Range

```text
-20 to 19
```

-20 = Highest Priority

19 = Lowest Priority

## Start Process with Low Priority

```bash
nice -n 10 command
```

Example:

```bash
nice -n 10 tar -czf backup.tar.gz /home
```

---

## What is Renice?

The `renice` command changes the priority of a running process.

## Syntax

```bash
renice priority PID
```

Example:

```bash
renice 5 1234
```

## Check Priority

```bash
ps -el
```

## Real-Life Use Case

Reduce CPU usage of backup jobs running in production servers.
