# Kill Command

## What is kill?

The `kill` command is used to terminate a process using its Process ID (PID).

## Syntax

```bash
kill PID
```

## Example

```bash
kill 1234
```

## Force Kill

```bash
kill -9 1234
```

## Find Process ID

```bash
ps -ef | grep nginx
```

## Common Signals

| Signal | Meaning |
|----------|----------|
| 1 | SIGHUP |
| 9 | SIGKILL |
| 15 | SIGTERM |

## Real-Life Use Case

Terminate a stuck application:

```bash
kill -9 PID
```
