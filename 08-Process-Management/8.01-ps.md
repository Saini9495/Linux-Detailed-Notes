# PS Command

## What is ps?

The `ps` command is used to display information about active processes running on the Linux system.

## Syntax

```bash
ps
```

## Common Examples

### Show current shell processes

```bash
ps
```

### Show all running processes

```bash
ps -e
```

### Detailed process information

```bash
ps -ef
```

### Search a specific process

```bash
ps -ef | grep nginx
```

## Important Columns

| Column | Description |
|----------|-------------|
| PID | Process ID |
| TTY | Terminal Type |
| TIME | CPU Time Used |
| CMD | Command Name |

## Real-Life Use Case

Find whether a service is running:

```bash
ps -ef | grep docker
```
