# Background and Foreground Processes

## Foreground Process

A process that runs directly in the terminal and occupies it until completion.

Example:

```bash
ping google.com
```

The terminal remains busy.

---

## Background Process

A process that runs in the background and frees the terminal.

Example:

```bash
ping google.com &
```

Output:

```bash
[1] 2345
```

2345 = PID

---

## View Background Jobs

```bash
jobs
```

---

## Move Process to Background

Press:

```bash
CTRL + Z
```

Then:

```bash
bg
```

---

## Bring Process to Foreground

```bash
fg
```

---

## Example Workflow

```bash
ping google.com
```

Press:

```bash
CTRL + Z
```

Move to background:

```bash
bg
```

Bring back:

```bash
fg
```

## Real-Life Use Case

Run long backup scripts while continuing other work in the terminal.
