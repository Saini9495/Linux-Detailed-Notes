# HTOP Command

## What is htop?

`htop` is an advanced and interactive version of the top command.

## Installation

Ubuntu:

```bash
sudo apt install htop
```

CentOS:

```bash
sudo yum install htop
```

## Run htop

```bash
htop
```

## Features

- Colored Interface
- Mouse Support
- Process Search
- Easy Process Management

## Useful Keys

| Key | Action |
|------|---------|
| F3 | Search Process |
| F5 | Tree View |
| F9 | Kill Process |
| F10 | Exit |

## Real-Life Use Case

Identify high CPU-consuming applications.
