# Linux Distributions

## What is a Distribution?

A Linux Distribution (Distro) is a complete operating system built using:

- Linux Kernel
- GNU Tools
- Package Manager
- Desktop Environment

Formula:

Linux Kernel + Software Packages = Linux Distribution

---

## Popular Linux Distributions

### Ubuntu

Best for:
- Beginners
- Developers
- DevOps Engineers

Package Manager:
apt

---

### Debian

Known for:
- Stability
- Reliability

Package Manager:
apt

---

### Fedora

Sponsored by Red Hat.

Used for:
- Latest Features
- Testing New Technologies

Package Manager:
dnf

---

### CentOS

Community version of Red Hat.

Popular in enterprise environments.

---

### Red Hat Enterprise Linux (RHEL)

Commercial Linux distribution.

Used in:
- Enterprises
- Data Centers

Package Manager:
yum/dnf

---

### Kali Linux

Used for:
- Ethical Hacking
- Penetration Testing

Contains hundreds of security tools.

---
