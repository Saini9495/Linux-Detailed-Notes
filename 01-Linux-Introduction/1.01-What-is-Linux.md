# What is Linux?

Linux is an open-source operating system kernel created by Linus Torvalds in 1991.

An Operating System (OS) acts as a bridge between users and computer hardware. Linux manages hardware resources such as CPU, Memory, Storage, and Network devices.

## Key Features

- Open Source
- Free to Use
- Multi-User
- Multi-Tasking
- Secure
- Stable
- Highly Customizable

## Real-Life Example

Think of Linux as the manager of a company:

- Employees → Users
- Machines → Hardware
- Manager → Linux OS

The manager ensures that every employee gets the resources they need without conflicts.

## Why Learn Linux for DevOps?

Almost all cloud servers run Linux.

Tools like:
- Docker
- Kubernetes
- Jenkins
- Ansible
- Terraform

are commonly deployed on Linux systems.

## Summary

Linux is a free, secure, and powerful operating system used by developers, system administrators, cloud engineers, and DevOps professionals worldwide.
