# Linux History

## Unix Era (1969)

Unix was developed at Bell Labs by:

- Ken Thompson
- Dennis Ritchie

Unix became one of the most influential operating systems.

---

## GNU Project (1983)

Richard Stallman started the GNU Project.

Goal:
Create a free Unix-like operating system.

GNU provided:

- Compiler (GCC)
- Shell
- Utilities

However, it lacked a kernel.

---

## Linux Kernel Creation (1991)

Linus Torvalds, a student from Finland, created the Linux Kernel.

Announcement:

"I am doing a free operating system as a hobby."

Initially Linux was just a kernel.

GNU tools + Linux Kernel = GNU/Linux

---

## Open Source Revolution

Developers worldwide contributed to Linux.

It became:

- Faster
- More Secure
- More Stable

---

## Modern Linux

Today Linux powers:

- AWS
- Google Cloud
- Azure
- Android
- Supercomputers
- Most Web Servers

## Timeline

1969 → Unix

1983 → GNU Project

1991 → Linux Kernel

1992 → GNU/Linux

2008 → Android

Present → Cloud & DevOps Dominance

## Summary

Linux started as a hobby project and became the backbone of modern cloud computing.
