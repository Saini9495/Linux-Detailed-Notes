# Arrays in Bash

## Create Array

```bash
fruits=("Apple" "Mango" "Banana")
```

## Access Elements

```bash
echo ${fruits[0]}
```

Output:

```text
Apple
```

## Print All Elements

```bash
echo ${fruits[@]}
```

## Array Length

```bash
echo ${#fruits[@]}
```

## Add Element

```bash
fruits+=("Orange")
```

## Loop Through Array

```bash
for fruit in "${fruits[@]}"
do
echo $fruit
done
```
