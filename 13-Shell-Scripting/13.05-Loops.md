# Loops in Bash

## For Loop

```bash
for i in {1..5}
do
echo $i
done
```

Output:

```text
1
2
3
4
5
```

## While Loop

```bash
count=1

while [ $count -le 5 ]
do
echo $count
((count++))
done
```

## Until Loop

```bash
count=1

until [ $count -gt 5 ]
do
echo $count
((count++))
done
```

## Break

```bash
for i in {1..10}
do
if [ $i -eq 5 ]
then
break
fi
echo $i
done
```

## Continue

```bash
for i in {1..5}
do
if [ $i -eq 3 ]
then
continue
fi
echo $i
done
```
