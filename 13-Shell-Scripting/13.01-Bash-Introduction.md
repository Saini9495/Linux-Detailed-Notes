# Bash Introduction

## What is Bash?

Bash (Bourne Again Shell) is a command-line interpreter used in Linux systems.

It acts as a bridge between the user and the operating system.

## Why Use Bash?

- Automate repetitive tasks
- Execute multiple commands together
- System administration
- Log management
- Server automation

## Check Bash Version

```bash
bash --version
```

## First Bash Script

Create a file:

```bash
nano hello.sh
```

Add:

```bash
#!/bin/bash

echo "Hello DevOps Engineers!"
```

Give execution permission:

```bash
chmod +x hello.sh
```

Run:

```bash
./hello.sh
```

Output:

```text
Hello DevOps Engineers!
```

## Shebang

```bash
#!/bin/bash
```

Tells Linux to execute the script using Bash.
