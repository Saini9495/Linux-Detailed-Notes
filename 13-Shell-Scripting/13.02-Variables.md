# Variables in Bash

Variables store data values.

## Creating Variables

```bash
name="Rishabh"
```

## Access Variable

```bash
echo $name
```

Output:

```text
Rishabh
```

## Multiple Variables

```bash
city="Delhi"
age=22

echo $city
echo $age
```

## User Input

```bash
read -p "Enter Name: " name

echo "Welcome $name"
```

## Environment Variables

```bash
echo $HOME
echo $USER
echo $PATH
```

## Constant Variable

```bash
readonly company="Google"
```
