# Functions in Bash

Functions help reuse code.

## Simple Function

```bash
greet() {
echo "Welcome to DevOps"
}

greet
```

## Function with Parameters

```bash
greet() {
echo "Hello $1"
}

greet Rishabh
```

Output:

```text
Hello Rishabh
```

## Return Value

```bash
sum() {
result=$((10+20))
echo $result
}

sum
```

## Advantages

- Code Reusability
- Better Maintenance
- Easy Debugging
