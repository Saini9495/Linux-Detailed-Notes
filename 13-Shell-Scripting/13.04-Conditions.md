# Conditional Statements

## if Statement

```bash
age=18

if [ $age -ge 18 ]
then
echo "Eligible"
fi
```

## if else

```bash
if [ $age -ge 18 ]
then
echo "Adult"
else
echo "Minor"
fi
```

## if elif else

```bash
marks=85

if [ $marks -ge 90 ]
then
echo "A Grade"

elif [ $marks -ge 70 ]
then
echo "B Grade"

else
echo "C Grade"
fi
```

## Case Statement

```bash
read choice

case $choice in
1) echo "Start";;
2) echo "Stop";;
3) echo "Restart";;
*) echo "Invalid";;
esac
```
