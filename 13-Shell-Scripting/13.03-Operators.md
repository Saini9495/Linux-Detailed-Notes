# Bash Operators

## Arithmetic Operators

```bash
a=20
b=10

echo $((a+b))
echo $((a-b))
echo $((a*b))
echo $((a/b))
```

## Comparison Operators

```bash
-eq
-ne
-gt
-lt
-ge
-le
```

Example:

```bash
if [ 10 -gt 5 ]
then
echo "True"
fi
```

## Logical Operators

AND

```bash
&&
```

OR

```bash
||
```

NOT

```bash
!
```

## Assignment Operators

```bash
=
+=
-=
```
