# Linux Log Files

## What are Log Files?

Log files are text files that store system events, application activities, errors, warnings, and security-related information.

Logs help administrators:

- Troubleshoot issues
- Monitor system health
- Detect security incidents
- Audit user activities

---

# Common Linux Log Locations

Most logs are stored inside:

```bash
/var/log
```

Check available logs:

```bash
ls /var/log
```

Example Output:

```bash
auth.log
syslog
kern.log
boot.log
dmesg
apache2/
nginx/
```

---

# Important Log Files

## 1. auth.log

Stores authentication-related logs.

```bash
cat /var/log/auth.log
```

Contains:

- SSH Login Attempts
- User Authentication
- sudo Commands

---

## 2. syslog

General system logs.

```bash
cat /var/log/syslog
```

Contains:

- System Messages
- Services Information
- Startup Events

---

## 3. kern.log

Kernel-related logs.

```bash
cat /var/log/kern.log
```

Contains:

- Hardware Detection
- Driver Events
- Kernel Messages

---

## 4. boot.log

System startup logs.

```bash
cat /var/log/boot.log
```

Contains:

- Boot Process Details
- Startup Errors

---

## 5. dmesg Log

Kernel ring buffer messages.

```bash
dmesg
```

Used for:

- Hardware Troubleshooting
- Driver Errors

---

# Reading Log Files

## View Entire File

```bash
cat /var/log/syslog
```

## View Last Lines

```bash
tail /var/log/syslog
```

## View Last 20 Lines

```bash
tail -20 /var/log/syslog
```

## Real-Time Monitoring

```bash
tail -f /var/log/syslog
```

---

# Search Logs

Using grep:

```bash
grep "error" /var/log/syslog
```

Case-insensitive:

```bash
grep -i "failed" /var/log/auth.log
```

---

# Log Rotation

Linux automatically rotates logs using:

```bash
logrotate
```

Configuration:

```bash
/etc/logrotate.conf
```

Check configuration:

```bash
cat /etc/logrotate.conf
```

---

# Best Practices

- Monitor logs regularly
- Archive old logs
- Enable log rotation
- Restrict log file access
- Centralize logs for large environments

Answer:
Most logs are stored in `/var/log`.

Q2: Which log stores SSH login attempts?

Answer:
`/var/log/auth.log`
