# Linux Monitoring Commands

## Introduction

Monitoring helps administrators track:

- CPU Usage
- Memory Usage
- Disk Usage
- Running Processes
- Network Activity

---

# 1. top

Real-time system monitoring.

```bash
top
```

Shows:

- CPU Usage
- Memory Usage
- Running Processes

Quit:

```bash
q
```

---

# 2. htop

Improved version of top.

Install:

```bash
sudo apt install htop
```

Run:

```bash
htop
```

Benefits:

- User Friendly
- Interactive
- Colored Interface

---

# 3. free

Check memory usage.

```bash
free -h
```

Example:

```bash
total used free
```

---

# 4. vmstat

System performance statistics.

```bash
vmstat
```

---

# 5. iostat

Disk performance monitoring.

Install:

```bash
sudo apt install sysstat
```

Run:

```bash
iostat
```

---

# 6. df

Disk space usage.

```bash
df -h
```

---

# 7. du

Directory size.

```bash
du -sh foldername
```

---

# 8. ps

View running processes.

```bash
ps aux
```

---

# 9. uptime

Check uptime and load.

```bash
uptime
```

Output:

```bash
10:00:00 up 5 days, load average: 0.25
```

---

# 10. netstat

Network monitoring.

```bash
netstat -tulnp
```

---

# 11. ss

Modern replacement for netstat.

```bash
ss -tulnp
```

---

# 12. lsof

Open files and ports.

```bash
lsof -i
```

---

# 13. sar

Historical performance data.

Install:

```bash
sudo apt install sysstat
```

CPU Report:

```bash
sar -u
```

Memory Report:

```bash
sar -r
```

---

# 14. watch

Run command repeatedly.

```bash
watch df -h
```

Updates every 2 seconds.

---
