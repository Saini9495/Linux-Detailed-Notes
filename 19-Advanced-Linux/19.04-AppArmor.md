# AppArmor

## What is AppArmor?

AppArmor is a Linux security framework.

Used to restrict application capabilities.

Popular In:
- Ubuntu
- Debian

## Why AppArmor?

Limits what applications can access.

Example:

Nginx can access:

✔ Web files

✘ User home directory

## Check Status

```bash
sudo aa-status
```

## AppArmor Modes

### Enforce

Rules applied.

### Complain

Logs violations only.

### Disable

AppArmor disabled.

## Profile Location

```bash
/etc/apparmor.d/
```

## Useful Commands

Enable Profile:

```bash
sudo aa-enforce profile
```

Complain Mode:

```bash
sudo aa-complain profile
```

Disable:

```bash
sudo aa-disable profile
```

## Advantages

- Easy Configuration
- Better Application Isolation
- Lightweight Security
