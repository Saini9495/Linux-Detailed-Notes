# Linux Kernel

## What is Kernel?

Kernel is the core of Linux Operating System.

Acts as bridge between:

Applications ↔ Hardware

## Responsibilities

- Process Management
- Memory Management
- Device Management
- File System Management
- Networking

## Kernel Architecture

User Space
↓
System Calls
↓
Kernel Space
↓
Hardware

## Check Kernel Version

```bash
uname -r
```

Detailed:

```bash
uname -a
```

## Kernel Types

### Monolithic Kernel

All services inside kernel.

Linux uses Monolithic Kernel.

### Micro Kernel

Minimal services inside kernel.

## Kernel Modules

Load Module:

```bash
modprobe module_name
```

View Loaded Modules:

```bash
lsmod
```

Module Information:

```bash
modinfo module_name
```

## Kernel Logs

```bash
dmesg
```

## Upgrade Kernel

Ubuntu:

```bash
sudo apt update
sudo apt upgrade
```

## Interview Questions

### What is kernel space?

Protected memory area used by kernel.

### What is user space?

Area where user applications run.
