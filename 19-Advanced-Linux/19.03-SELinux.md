# SELinux (Security Enhanced Linux)

## What is SELinux?

SELinux is a Linux security module that provides mandatory access control (MAC).

Developed by:
- NSA
- Red Hat

## Why SELinux?

Even if a hacker gains root access:

SELinux can restrict actions.

## SELinux Modes

### Enforcing

Security rules are applied.

### Permissive

Rules are checked but not enforced.

### Disabled

SELinux turned off.

## Check SELinux Status

```bash
sestatus
```

or

```bash
getenforce
```

## Change Mode

Temporary:

```bash
setenforce 0
```

Permanent:

Edit:

```bash
/etc/selinux/config
```

```bash
SELINUX=enforcing
```

## File Context

View Context:

## Interview Questions

### Difference Between DAC and MAC?

DAC:
Owner controls permissions.

MAC:
System controls permissions.

SELinux uses MAC.
