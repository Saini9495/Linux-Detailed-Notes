# Linux Performance Tuning

## What is Performance Tuning?

Process of improving:

- CPU Performance
- Memory Usage
- Disk I/O
- Network Performance

## System Monitoring Tools

### CPU Usage

```bash
top
```

```bash
htop
```

### Memory Usage

```bash
free -h
```

### Disk Usage

```bash
df -h
```

### Disk I/O

```bash
iostat
```

### Process Monitoring

```bash
ps aux
```

## CPU Optimization

Find High CPU Process:

```bash
top
```

Kill Process:

```bash
kill -9 PID
```

## Memory Optimization

Clear Cache:

```bash
sync
echo 3 > /proc/sys/vm/drop_caches
```

## Disk Optimization

Check Large Files:

```bash
du -sh *
```

Find Large Directories:

```bash
du -h --max-depth=1
```

## Network Optimization

Check Open Ports:

```bash
ss -tulnp
```

Network Statistics:

```bash
netstat -an
```

## Load Average

```bash
uptime
```

Example:

```bash
load average: 0.20, 0.40, 0.60
```

Meaning:

- 1 min load
- 5 min load
- 15 min load

## Performance Troubleshooting Flow

High CPU
↓
top

High Memory
↓
free -h

Disk Full
↓
df -h

Disk Slow
↓
iostat

Network Slow
↓
ss / netstat
