# VS Code Remote Development

## Introduction

Visual Studio Code allows developers to connect directly to remote Linux servers and edit files without logging into the server terminal manually.

## Benefits

- Remote file editing
- Integrated terminal
- SSH support
- Easy development workflow

## Prerequisites

- VS Code Installed
- Remote SSH Extension
- Linux Server
- SSH Access

## Install Remote SSH Extension

1. Open VS Code
2. Go to Extensions
3. Search:

```text
Remote - SSH
```

4. Install Extension

## Connect to Linux Server

Press:

```text
Ctrl + Shift + P
```

Select:

```text
Remote-SSH: Connect to Host
```

Enter:

```bash
ubuntu@your-server-ip
```

Example:

```bash
ubuntu@54.123.45.67
```

## SSH Key Authentication

Generate SSH Key:

```bash
ssh-keygen
```

Copy Key:

```bash
ssh-copy-id ubuntu@server-ip
```

Connect:

```bash
ssh ubuntu@server-ip
```

## Working with Files

Once connected:

- Open folders directly from server
- Edit files remotely
- Save changes instantly
- Access terminal inside VS Code

## Example

Edit Nginx Configuration:

```bash
/etc/nginx/nginx.conf
```

Edit Docker Compose File:

```bash
docker-compose.yml
```

Edit Application Files:

```bash
app.py
```
