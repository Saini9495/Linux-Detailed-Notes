# Vim Editor

## Introduction
Vim (Vi Improved) is a powerful command-line text editor used in Linux systems. It is widely used by Linux Administrators, DevOps Engineers, and Developers.

## Why Vim?
- Fast and lightweight
- Available on almost every Linux distribution
- Supports keyboard-based editing
- Highly customizable

## Open a File

```bash
vim filename.txt
```

## Vim Modes

### 1. Normal Mode
Default mode used for navigation.

### 2. Insert Mode
Used for writing text.

Press:

```bash
i
```

### 3. Command Mode
Used for saving and quitting.

Press:

```bash
:
```

## Common Commands

### Save File

```bash
:w
```

### Save and Exit

```bash
:wq
```

### Exit Without Saving

```bash
:q!
```

### Delete a Line

```bash
dd
```

### Copy a Line

```bash
yy
```

### Paste

```bash
p
```

### Undo

```bash
u
```

### Search Text

```bash
/text
```

## Navigation Commands

| Command | Description |
|----------|------------|
| h | Left |
| l | Right |
| j | Down |
| k | Up |
| gg | First Line |
| G | Last Line |

## Advantages
- Extremely fast
- Minimal resource usage
- Best for server administration

## Summary
Vim is the most commonly used Linux text editor for server management and DevOps tasks.
