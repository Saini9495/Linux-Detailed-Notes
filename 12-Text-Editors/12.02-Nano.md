# Nano Editor

## Introduction

Nano is a beginner-friendly command-line text editor available in Linux systems.

## Why Nano?

- Easy to learn
- User-friendly
- Commands visible at bottom
- Perfect for beginners

## Open File

```bash
nano filename.txt
```

## Create New File

```bash
nano newfile.txt
```

## Common Shortcuts

| Shortcut | Description |
|-----------|------------|
| CTRL + O | Save File |
| CTRL + X | Exit |
| CTRL + K | Cut Line |
| CTRL + U | Paste |
| CTRL + W | Search |
| CTRL + C | Show Cursor Position |

## Editing Example

```bash
nano notes.txt
```

Add text:

```text
Hello Linux
Welcome to Nano Editor
```

Save:

```bash
CTRL + O
```

Exit:

```bash
CTRL + X
```

## Advantages

- Beginner Friendly
- Easy Navigation
- No Need to Learn Modes

## Disadvantages

- Less powerful than Vim
- Limited customization

## Summary

Nano is best for beginners who want a simple and easy text editor.
